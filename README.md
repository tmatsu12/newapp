# Ruraloos（ルーラルーズ）

## サイト概要
地方移住を希望する方々の情報共有サイト

### サイトテーマ
地方移住を考えている方々が、移住してみたい地方（Rural）について正しい情報を知り、
カンガルー（Kangaroo）のように勇気を持って勢いよく飛び出していくことを後押しすることを目的としています。
